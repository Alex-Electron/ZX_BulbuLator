## Плата: EBAZ4205

EBAZ4205 — это плата на базе Zynq-7000, извлечённая из оборудования для майнинга биткойнов компании Ebang. Она дешёвая и её легко найти — именно поэтому её и выбрали для этого проекта.

Целевой версией является **Zynq-7010** (`XC7Z010`) — самая распространённая на рынке подержанного оборудования, поэтому разработка и тестирование именно на ней позволяют обеспечить соответствие проекта тому, что есть у большинства людей. Всё здесь рассчитано на `xc7z010clg400-1`.

Есть также платы с **специально впаянным `XC7Z020`** вместо 7010 — это та же плата, но с большим объёмом программируемой логики. Такие тоже можно купить; они стоят дороже, так как установка более крупного чипа — это доработка, а не то, с чем эти платы поставлялись изначально. (У меня тоже есть такая.) Корпус у них одинаковый (`clg400`), так что распиновка совпадает, и сборка для 7020 сводится просто к замене `-part`.

PetaLinux 2024.1 уже загружается на этой плате.

## Ресурсы платы

EBAZ4205 — это плата управления для майнинга, так что за такую цену она отлично укомплектована:

| | EBAZ4205 |
|---|---|
| SoC | Xilinx Zynq-7000 `XC7Z010` (`clg400`, speed grade -1) |
| Процессор (PS) | двухъядерный ARM Cortex-A9, 666 МГц |
| DDR3 (на PS) | 256 МБ |
| NAND-флеш | 128 МБ |
| Ethernet | 10/100, PHY IP101GA (кварц 25 МГц) |
| microSD | слот на плате — устройство загрузки для этого проекта |
| Опорная частота PS | 33,33 МГц; дизайн поднимает тактовую частоту ПЛИС 100 МГц (FCLK0) |
| Питание | 5–12 В |
| Заводская загрузка | NAND; здесь перестроена на SD |
| Светодиоды | два на плате (используются для мигалки на этапе bring-up) |

Именно в программируемой логике и проявляется важность выбора микросхемы. XC7Z010 — это небольшой компонент, а плотная сборка ZX уже полностью заполняет его Block RAM:

| Программируемая логика | `XC7Z010` (цель) | `XC7Z020` (доработанная плата) |
|---|---|---|
| Логические ячейки | 28 000 | 85 000 |
| LUT | 17 600 | 53 200 |
| Триггеры | 35 200 | 106 400 |
| Block RAM | 60 × 36 Кбит (2,1 Мбит) | 140 × 36 Кбит (4,9 Мбит) |
| DSP-блоки | 80 | 220 |

Оперативная память Spectrum находится в этом блоке BRAM (в текущей сборке используется 60/60), поэтому перенос ОЗУ с внешнего контроллера SDRAM на встроенный BRAM здесь так важен. Вместо этого 256 МБ DDR3 передаёт фреймбуфер с тройным буфером по интерфейсу AXI. На платах 7020 все показатели ПЛИС примерно утраиваются, что и составляет необходимый запас мощности для 16-битной машины — но корпус и распиновка остаются такими же, так что сборка 7020 сводится лишь к замене `-part`.

## Плата расширения

Есть специальная плата расширения HDMI / аудио, которая уже собрана и работает. Она выдает чистый сигнал в формате PAL с частотой 50 Гц (720×576@50Hz) без мерцания и поддерживает аудиоканал I²S.

## Программирование и JTAG

Более объёмные битстримы размером 128K не удавалось надёжно прошить через обычный XVC; процесс постоянно срывался из-за ошибок «bad-packet». Решением стала Raspberry Pi Pico с пользовательской прошивкой XVC, настроенной на медленную скорость нарастания сигнала и силу тока 2 мА. С этой прошивкой ядра размером 128K прошиваются через JTAG / XVC за пару секунд с первой попытки.

Синтез и bootgen (Vivado, Vitis, JTAG-демоны) запускаются на ThinkPad. Прошивка происходит локально через Pico.
