# Исследования

Languages: [English](README.md) · **Русский**

Заметки по наладке и эксперименты с EBAZ4205 (Zynq-7010) — каждый шаг представляет собой самостоятельную часть с исходными кодами и готовым к записи битстримом. Это те строительные блоки, на которых построены ядра BulbuLator, записанные по мере их проверки на реальном оборудовании.

| Этап | Что он включает |
|------|----------------|
| [00 — Настройка и подключение](00-setup/) | С «голой» платы: питание, перемычка режима загрузки, JTAG-программатор (реальный кабель или Pico), установка ПО и запись битстрима |
| [01 — Запуск платы: мигание светодиодов](01-board-bringup-blink/) | Питание, канал JTAG → PL и работающий битстрим, использующий только внутренний генератор микросхемы |
| [02 — Управление светодиодами с помощью кнопок](02-buttons-and-leds/) | Считывание сигналов с четырёх кнопок на шилде (активный низкий уровень, синхронизированные) и их использование для управления светодиодами |
| [03 — Простой HDMI: цветовые полосы](03-hdmi-bars/) | Первое изображение на экране — восемь цветовых полос с разрешением 720p, синхронизируемых с PS (урок по преобразователю уровней / `ps7_post_config`) |
| [04 — HDMI с узорами, переключаемыми кнопками](04-hdmi-buttons/) | Прыгающий квадрат и четыре узора (полосы, градиент, шахматная доска), между которыми переключают кнопки шилда |
| [05 — Аудио по HDMI: квадратик пищит](05-hdmi-beep/) | Первый звук — прыгающий квадратик пищит через аудио HDMI, когда упирается в стенку, с помощью ядра hdl-util/hdmi |
| [06 — ZX Spectrum 128](06-zx-spectrum-128/) | Всё сложилось: настоящий ZX Spectrum 128 на плате — видео и аудио по HDMI, кнопки шилда управляют меню загрузки, он загружает игры с кассеты через разъём и запускается с SD-карты. Построен на ядре Atlas `zx` |
| [07 — Пробуждение ARM: плоскость управления PS↔PL](07-arm-control-plane/) | Интерфейс регистров AXI, позволяющий находящемуся в режиме ожидания процессору ARM останавливать Z80 и читать/записывать в память Spectrum в режиме реального времени — основа для загрузки игр с SD-карты. Чистый супернабор от Шаг 6 |
| [08 — Видео без рывков: фреймбуфер DDR с двойным буфером](08-ddr-framebuffer/) | Буферизует кадр ZX объёмом 51 КБ в PS DDR (тройной буфер, с переключением во время vblank HDMI), чтобы в демо с эффектом бордюров и при переворачивании теневого экрана не было разрывов — первое реальное использование канала AXI-HP для доступа к DDR |

Шаги 0–5 — это стартовая полоса; шаг 6 — первая реальная машина. С этого момента инструкции по запуску превращаются в портирование ядер (большая точность, больше демо и, в конечном итоге, более мощные машины, которым нужен PS DDR).
